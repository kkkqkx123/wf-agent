# PDF Processing

This skill provides guidance for working with PDF documents.

## Common Operations

### Reading PDFs

- Extract text content from PDFs
- Read metadata and document properties
- Identify form fields and annotations

### Editing PDFs

- Fill in form fields
- Add or modify annotations
- Rotate pages
- Add watermarks

### Merging PDFs

- Combine multiple PDFs into one
- Maintain page order and bookmarks
- Handle different page sizes

### Splitting PDFs

- Extract specific pages
- Split by bookmarks or page ranges
- Create separate files for each page

### Converting PDFs

- PDF to images (PNG, JPEG)
- Images to PDF
- PDF to text
- PDF to Word/HTML

## Tools and Libraries

### Python

- PyPDF2: Basic PDF operations
- pdfplumber: Text extraction with layout
- reportlab: PDF generation
- PyMuPDF: Advanced operations

### Node.js

- pdf-lib: PDF creation and modification
- pdf-parse: Text extraction
- pdf2pic: PDF to image conversion

## Best Practices

1. Always backup original files before editing
2. Validate PDF structure after operations
3. Handle encrypted PDFs appropriately
4. Consider file size optimization
5. Test with various PDF types (scanned, digital, forms)
